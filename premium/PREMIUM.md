## Sample Recipes

### Recipe 001: Meeting Notes to Action Items
```
/recipe-001 Today's standup: John mentioned API is delayed.
Sarah completed login. Demo pushed to Friday.
```
**Output:** Formatted summary, action items table with owners/deadlines, follow-up emails.

### Recipe 013: Performance Review Writing
```

Achievements: Led migration project, mentored 2 juniors

```
**Output:** Complete performance review with specific examples, ratings guidance, development goals.

```
Users: Power users who check daily
```
**Output:** Full PRD with user stories, acceptance criteria, technical considerations.
